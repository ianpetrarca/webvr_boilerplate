Build WebXR apps with Aframe, Webpack and Firebase.

WebXR Boilerplate


WebXR Boilerplate is a toolkit for XR creators to develop, build and deploy static XR experiences to the web. WebXR Boilerplate comes with a hot-reload webserver, the latest builds of Aframe, Aframe's best community components and a Webpack build pipeline that packages your content for Firebase Hosting.

Github Link: https://github.com/ianpetrarca/webxr_boilerplate

WebXR Demo Link: https://ianpetrarca.com/webxrboilerplate
